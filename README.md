# ARCHEOLOGY
 ## Visão Geral
Este projeto tem como objetivo explorar conceitos de Realidade Aumentada (AR) permitindo que o usuário interaja com formas geométricas virtuais, como quadrados e círculos, posicionadas no espaço real por meio da câmera do dispositivo.
O sistema possibilita a movimentação desses objetos no ambiente e a identificação do tipo de forma através de um processo de escaneamento.


## Objetivo do Projeto

- Proporcionar uma experiência interativa em realidade aumentada
- Permitir a interação de objetos no espaço físico
- Identificar o tipo de objeto escaneado

## Tecnologias utilizadas

- C#
- Unity 6.2
- AR Foundation

## Instalando no seu dispositivo

- Clique em Build Settings
- Selecione a opção desejada entre as plataformas disponíveis (caso necessário, instale os módulos para o build)
- Clique em Switch Platform para selecionar
- Clique em Player Settings caso queira alterar o nome do projeto no dispositivo
- Clique em Build e selecione a pasta onde será salvo o build
- Após finalizar o carregamento, será gerada uma pasta com os arquivos a serem colocados no dispositivo
-Basta realizar a transferência dos arquivos de acordo com o sistema utilizado e executar no seu dispositivo
